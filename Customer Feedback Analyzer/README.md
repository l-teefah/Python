# Customer Feedback Analyzer

## Overview
This folder contains a comprehensive analysis project for processing and analyzing customer feedback data. The project uses Jupyter notebooks to explore guest reviews and extract meaningful insights from customer feedback.

## Contents

### Files
- **customer_feedback_analyzer_notebook.ipynb** - Main Jupyter notebook containing the complete analysis workflow, including data loading, preprocessing, sentiment analysis, and visualization of guest feedback patterns.
- **guest_data_with_reviews.xlsx** - Excel file containing guest data and associated customer reviews used as the data source for analysis.

## Purpose
This project analyzes customer feedback to:
- Extract insights from guest reviews
- Identify sentiment patterns in customer feedback
- Analyze common themes and topics mentioned in reviews
- Visualize feedback trends and distributions

## Technology Stack
- **Language**: Python (Jupyter Notebook)
- **Libraries**: Common data analysis libraries (pandas, numpy, matplotlib, seaborn, etc.)
- **Data Format**: Excel (.xlsx)

## Getting Started
1. Ensure you have Jupyter notebook installed
2. Open `customer_feedback_analyzer_notebook.ipynb`
3. The notebook will load the data from `guest_data_with_reviews.xlsx`
4. Run cells sequentially to execute the analysis

## Key Features
- Data exploration and cleaning
- Sentiment analysis of reviews
- Pattern recognition in customer feedback
